페이스북 메신져(Messenger) 연동을 위해서 아래의 항목을 체크해주세요

  • 1) 페이스북 개발자센터(https://developers.facebook.com/) 접속.
  • 2) 메신져를 연동할 특정 app 혹은 신규 app 대시보드 페이지 접속.
  • 3) 대시보드 페이지에서 제품추가 버튼 클릭한 후 Messenger 제품 의 설정버튼 클릭해서 설정 페이지((/apps/앱ID/messenger/settings)로 이동.
  • 4) 3)의 메신져 설정 페이지 항목 중 Webhooks 항목의 Webhooks 설정 버튼 클릭.
  • 5) 팝업창에 나오는 설정항목 중 콜백URL 과 확인토큰 입력란에 아래의 값을 각각 입력하고 받아보기 필드 항목의 내용중 messages, messaging_postbacks 를 체크(개발상황에 따라 추후 추가항목 체크할 수 있음)한 후 확인 및 저장 버튼 클릭.
  • 콜백 URL
  • 확인 토큰
  • 6) 3)의 메신져 설정 페이지 항목 중 토큰 생성 항목에서 메신져를 연동할 페이지 선택하면 페이지 토큰이 생성되며 이 페이지 토큰 값을 아래 입력란에 입력하고 아래 저장버튼 클릭 .
  • 페이지 엑세스 토큰
