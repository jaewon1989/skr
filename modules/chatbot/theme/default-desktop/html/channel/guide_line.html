라인(LINE) 메시징API 연동을 위해서 아래의 항목을 체크해주세요

  • 1) 라인 모바일앱을 실행(없는 경우 다운로드).
  • 2) 라인앱 우상단 설정메뉴를 클릭하고 계정 메뉴 클릭.
  • 3) 계정설정 페이지 항목 중 이메일 과 비밀번호 항목 등록한 후 하단 로그인 허용 항목 활성화(체크).
  • 4) 라인 메시징API 관리자센터(https://developers.line.me/) 접속 후 하단의 Messaging API 메뉴 클릭.
  • 5) Messaging API 페이지에서 Start now 버튼 클릭 후 3) 에서 등록한 이메일/비밀번호 로 로그인.
  • 6) 해당 절차에 따라 Provider 및 Channel 을 생성한 후 설정페이지(이하 7,8,9번 항목진행)로 이동.
  • 7) Use webhooks 항목을 활성화(Enabled) 변경(체크).
  • 8) 아래의 URL을 Webhook URL 항목 입력란에 입력(https:// 포함여부 체크).
  • 9) Channel Access Token 을 발급(Issue)한 후 Channel ID, Channel Secret, Access Token 을 아래의 입력란에 입력.
  • Channel ID
  • Channel Secret
  • Access Token
