네이버톡톡 연동을 위해서 아래의 항목을 체크해주세요

  • 1) 네이버톡톡 파트너센터(https://partner.talk.naver.com/) 접속.
  • 2) 회사 아이디 혹은 대표 아이디로 로그인.
  • 3) 계정관리 페이지(https://partner.talk.naver.com/) 로 이동.
  • 4) 기존 계정을 사용할 경우 바로 계정관리 페이지로 이동하고('계정관리' 버튼) 신규 계정 생성.
  • 5) 계정관리 페이지(https://partner.talk.naver.com/{고유번호}) 메뉴 중 '개발자도구 > 챗봇API 설정' 메뉴 클릭.
  • 6) Webhook > 이벤트 받을 URL 에 아래 URL 을 등록(저장)하고 Event 변경 버튼 클릭해서 send,open 2개의 이벤트를 선택.
  • 7) 보내기 API > Authorization 에서 인증코드를 발급받은 후 아래 입력란에 입력.
